Sobre este Proyecto

Este es mi proyecto de Django para 2º DAW. Es un espacio personal donde guardar contenido inspiracional: memes, citas, imágenes, lo que sea.

Funcionalidades

  • Guarda contenido con texto y/o imagen
  • Organiza por temas/tags
  • Marca tus favoritos
  • Bandeja Inbox para contenido sin clasificar
  • Vista aleatoria para inspirarte
  • Búsqueda y filtros

Tecnologías usadas

  • Django 5.0
  • SQLite
  • Bootstrap 5
  • Crispy Forms
